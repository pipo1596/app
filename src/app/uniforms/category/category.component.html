When building a website, we organize items into categories to make them easy for customers to find. Think of categories like folders in a filing cabinet - each folder can contain other folders or files, but the actual products (the "files") can only live inside the smallest folders.

Products can only be added to the lowest level of the hierarchy - the categories that do not have any sub-categories inside them.
